# ringcentral-python

RingCentral Python Client library


## Setup

```
pip install -Ur requirements.txt
```


## How to test

Create `.env` file with the following content:

```
production=false
server=https://platform.devtest.ringcentral.com
appKey=appKey
appSecret=appSecret
username=username
extension=extension
password=password
receiver=number-to-receiver-sms
```

Run `python main.py`
